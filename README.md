# 3D Tank Battle

基于 Three.js 的 3D 坦克大战游戏，单文件运行，无需安装。

## 在线试玩

**[点击这里开始游戏](https://wshuyi.github.io/tank-battle-3d/)**

## 操作方式

| 按键 | 功能 |
|------|------|
| WASD | 移动坦克 |
| Q/E | 旋转炮塔 |
| 空格 | 开火 |
| R | 修理（消耗 50 分） |
| Enter | 开始/重玩 |

## 游戏特性

- 多波次敌人，难度递增
- AI 巡逻、追击、攻击行为
- 实时小地图
- 生命值与修复系统

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 技术栈

- Three.js r128
- 纯 JavaScript，无构建依赖
